A structured and beginner-friendly repository to learn **Pandas** — one of the most essential Python libraries for data analysis and data science.

This repo covers everything from the basics of Series and DataFrames to data cleaning, groupby operations, aggregation, and data visualization.

---

## 🎯 Purpose of this Repository

- Learn Pandas step-by-step from scratch
- Understand Series and DataFrame structures
- Clean and preprocess real-world data
- Perform groupby and aggregation operations
- Visualize data insights using graphs
- Build a strong base for Data Science and Machine Learning

---

## 📚 Topics Covered

| File No. | Topic | Description |

| 01 | **Intro & Series** | Introduction to Pandas, installing, importing, creating and working with Series |

| 05 | **Visualization** | Plotting directly from Pandas — line, bar, histogram, scatter plots |

## 🧠 What You Will Learn

### 📌 1. Intro & Series
- What is Pandas and why use it?
- Installing and importing Pandas
- Creating a Series from list, dict, and array
- Series indexing, slicing, and attributes
- Basic Series operations

### 📌 2. DataFrame Basics
- Creating DataFrames from dict, list, CSV
- Viewing data — `head()`, `tail()`, `info()`, `describe()`
- Selecting columns and rows
- `loc[]` and `iloc[]` indexing
- Adding and dropping columns

### 📌 3. Data Cleaning
- Detecting and handling missing values (`NaN`)
- Filling and dropping null values
- Removing duplicate rows
- Renaming columns and index
- Changing data types
- Replacing and mapping values

### 📌 4. Groupby & Aggregation
- `groupby()` — grouping data by columns
- Aggregate functions — `sum()`, `mean()`, `count()`, `min()`, `max()`
- Multiple aggregations with `agg()`
- Pivot tables
- `apply()` and `transform()`

### 📌 5. Visualization
- Plotting directly from a DataFrame
- Line plots, bar charts, histograms
- Scatter plots and box plots
- Customizing plots — titles, labels, colors
- Combining Pandas with Matplotlib

## ⚙️ How to Use This Repo

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/Pandas.git
   ```
2. Install required libraries
   ```bash
   pip install pandas matplotlib jupyter
   ```
3. Open Jupyter Notebook
   ```bash
   jupyter notebook
   ```
4. Start from `01_intro_and_series.ipynb` and go topic by topic
5. Run the code cells, experiment, and modify
